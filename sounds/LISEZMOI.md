# Dossier des sons 🔊

Les sons sont joués par **l'écran de salle** (le PC branché à la sono).
Déposer ici des fichiers **.mp3** portant EXACTEMENT ces noms — n'importe quel son peut être
remplacé sans toucher au code (ou directement depuis l'app : ✏️ Édition → 🔊 Sons).
Un fichier absent = pas de son (l'app fonctionne quand même).

| Fichier | Joué quand… | Note |
|---|---|---|
| `attente.mp3` | écran d'accueil affiché | **en boucle** — musique d'ambiance d'attente |
| `debut.mp3` | début de soirée (accueil → manche 1) | |
| `tirage.mp3` | un numéro est tiré (~90 fois par grille !) | TRÈS discret |
| `valid.mp3` | vérification : le numéro est bien sorti | « ding » doux |
| `rate.mp3` | vérification : le numéro n'est PAS sorti | le faux espoir |
| `suspense.mp3` | vérification en mode suspense | **en boucle** |
| `gagne.mp3` | verdict GAGNÉ | |
| `fauxbingo.mp3` | verdict FAUX BINGO | |
| `entracte.mp3` | lancement d'un entracte | |
| `reprise.mp3` | reprise de partie après l'entracte | si absent : son d'entracte |
| `fin.mp3` | fin de soirée (écran de remerciements) | |

Taille conseillée : jingles courts ; max 700 Ko pour le remplacement depuis l'app.
Volume : viser des fichiers déjà équilibrés entre eux (le réglage fin se fait sur la sono).
